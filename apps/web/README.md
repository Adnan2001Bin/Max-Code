# Web App (Next.js)

Frontend for the Quran Reader project.

## Key features

- Surah listing page (all 114 surahs)
- Static Surah pages with ayah reading view
- Translation search page (API-backed)
- Responsive settings sidebar:
  - Arabic font selection (Noto Naskh Arabic, Amiri)
  - Arabic font size control
  - Translation font size control
  - localStorage persistence

## Local development

```bash
npm run dev --workspace web
```
